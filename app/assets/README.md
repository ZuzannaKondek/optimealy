# Assets Directory

This directory should contain:
- `icon.png` - App icon (1024x1024px recommended)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon
- `favicon.png` - Web favicon

For now, placeholder files are created. Replace them with actual assets later.

## Quick Fix for Development

If you get "Asset not found" errors, you can temporarily remove the icon requirement from `app.config.js` or create simple placeholder images.
